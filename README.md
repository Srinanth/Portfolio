# Srinanth M V - Portfolio

Welcome to my personal portfolio repository! This is a modern, interactive, and responsive web portfolio built with **React**, **Tailwind CSS**, **Framer Motion**, and other modern web technologies. It showcases my skills, education, projects, and provides an interactive contact experience.

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS  
- **Animations:** Framer Motion, AniJS animations  
- **Icons:** Lucide React, React Icons  
- **Forms & Notifications:** EmailJS, react-hot-toast
